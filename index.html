QQ Widget for JAVA


  • 版本Version 1.01 | 更新时间20200618
  • 下载demo查看兼容性与其效果
  • 下载com.tencent.qq.widget.jar放置项目下libs内
  • 下载anim.zip解压后复制到项目/res/anim/下

  • 开始使用

导入Widget组件

  1. import com.tencent.qq.widget.*;
  2. import com.tencent.qq.widget.QQMenuDialog.*;
  • 调用方法

一个按钮的对话框

  1. final QQDialog dialogs=new QQDialog(MainActivity.this,R.style.dialog_qq_animation);//定义一个对话框及弹出动画
  2. dialogs.setLineColor(QQDialog.Colors.ORANGE );//设置对话框顶部线条的颜色
  3. dialogs.setTitle("标题",QQDialog.setTextColor.RED);//设置对话框的标题
  4. dialogs.setMessage("这是一个按钮的对话框",QQDialog.setTextColor.BROWN);//设置对话框的内容
  5. dialogs.setCanceledOnTouchOutside(true);//设置对话框是否可以触摸关闭
  6. dialogs.setCancelable(true);//设置对话框是否可以点击返回键关闭
  7. dialogs.setPositiveButton("确定",QQDialog.setTextColor.BLUE,new OnClickListener()
  8.               {
  9.                   @Override
  10.                   public void onClick(View p1)
  11.                   {   //点击事件
  12.                       dialogs.dismiss();//关闭对话框
  13.                   }
  14.               } );
  15. dialogs.show();//显示对话框

简化方法

  1. new QQDialog(MainActivity.this).setLineColor(QQDialog.Colors.BLUE).setTitle("标题").setMessage("这是两个按钮的对话框").setPositiveButton("确定",null).show();
